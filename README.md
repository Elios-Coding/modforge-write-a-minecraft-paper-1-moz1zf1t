# Write a Minecraft Paper 1 - Secret Menu XRAY

Client-side Fabric mod that listens for the swap-hands key and shows a simple Secret Menu overlay with an XRAY toggle; when enabled, it tries to hide common blocks in the local client's render for a crude x-ray effect.

Generated by **ModForge AI**.

- Minecraft: `1.21.11`
- Loader: Fabric `0.18.4`
- Fabric API: `0.141.3+1.21.11`

## Build

```bash
./gradlew build
```

The compiled `.jar` will be placed in `build/libs/`. GitHub Actions builds it automatically on push and publishes it to Releases.
